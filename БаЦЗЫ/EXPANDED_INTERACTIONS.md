# 🔗 РАСШИРЕННЫЕ ВЗАИМОДЕЙСТВИЯ СТОЛПОВ

## 📋 Что добавлено

### ✅ 1. Все 12 пар наказаний (刑)

**Было:** 6 основных пар  
**Стало:** Все 12 пар, включая самонаказания

#### Тройное наказание (三刑):
- **寅刑巳, 巳刑申, 申刑寅** — циклическое наказание между Деревом, Огнём и Металлом
- **丑刑戌, 戌刑未, 未刑丑** — циклическое наказание Земли

#### Самонаказание (自刑):
- **辰自刑** — самонаказание Дракона (внутренние противоречия)
- **午自刑** — самонаказание Лошади (излишняя самокритика)
- **酉自刑** — самонаказание Петуха (перфекционизм)
- **亥自刑** — самонаказание Свиньи (склонность к самокопанию)

#### Дополнительные пары:
- **子刑卯, 卯刑子** — наказание между Водой и Деревом

---

### ✅ 2. Все 12 пар вреди (害)

**Было:** 6 основных пар  
**Стало:** Все 12 пар (каждая пара в обе стороны)

#### Полный список:
- 子害未 / 未害子 — Вода ↔ Земля
- 丑害午 / 午害丑 — Земля ↔ Огонь
- 寅害巳 / 巳害寅 — Дерево ↔ Огонь
- 卯害辰 / 辰害卯 — Дерево ↔ Земля
- 申害亥 / 亥害申 — Металл ↔ Вода
- 酉害戌 / 戌害酉 — Металл ↔ Земля

---

### ✅ 3. Взаимодействия небесных стволов (合化)

**Добавлено:** Анализ слияний стволов с трансформацией в другие элементы

#### 5 основных пар трансформации:
- **甲己合化土** — Дерево Ян + Земля Инь → Земля (усиливает стабильность)
- **乙庚合化金** — Дерево Инь + Металл Ян → Металл (усиливает решимость)
- **丙辛合化水** — Огонь Ян + Металл Инь → Вода (усиливает мудрость)
- **丁壬合化木** — Огонь Инь + Вода Ян → Дерево (усиливает рост)
- **戊癸合化火** — Земля Ян + Вода Инь → Огонь (усиливает энтузиазм)

**Особенность:** Когда два ствола сливаются, они могут трансформироваться в третий элемент, усиливая его влияние в карте.

---

### ✅ 4. Специальные комбинации (三合, 三会)

**Добавлено:** Анализ мощных триад ветвей

#### Тройное слияние (三合) — 4 триады:
- **申子辰三合水** — Обезьяна + Крыса + Дракон → мощная энергия Воды
- **亥卯未三合木** — Свинья + Кролик + Коза → мощная энергия Дерева
- **寅午戌三合火** — Тигр + Лошадь + Собака → мощная энергия Огня
- **巳酉丑三合金** — Змея + Петух + Бык → мощная энергия Металла

#### Тройное собрание (三会) — 4 триады по сезонам:
- **寅卯辰三会木** — Весна (Тигр + Кролик + Дракон) → мощная энергия Дерева
- **巳午未三会火** — Лето (Змея + Лошадь + Коза) → мощная энергия Огня
- **申酉戌三会金** — Осень (Обезьяна + Петух + Собака) → мощная энергия Металла
- **亥子丑三会水** — Зима (Свинья + Крыса + Бык) → мощная энергия Воды

**Особенность:** 
- Полная триада (все 3 ветви) — очень сильное влияние
- Частичная триада (2 из 3) — умеренное влияние

## 🚀 Использование

### В результате анализа:

```javascript
const baziAnalysis = getFullBaziAnalysis(...);

// Взаимодействия ветвей (расширенные)
baziAnalysis.interactions // Все 12 пар наказаний и вреди

// Взаимодействия стволов
baziAnalysis.stemInteractions // Слияния стволов с трансформацией

// Специальные комбинации
baziAnalysis.specialCombinations // Триады (三合, 三会)
```

### Структура взаимодействий стволов:

```javascript
{
  type: '合化',
  name: '甲己合化土',
  pillars: ['年柱', '月柱'],
  transformsTo: 'Земля',
  description: 'Слияние Дерева Ян и Земли Инь трансформируется в Землю...',
  impact: 'positive'
}
```

### Структура специальных комбинаций:

```javascript
{

  pillars: ['年柱', '月柱', '日柱'],
  element: 'Вода',
  season: 'Зима', // только для 三会
  description: 'Тройное слияние создаёт мощную энергию Воды...',
  impact: 'positive',
  completeness: 'complete' // или 'partial'
}
```
